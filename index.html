PROFE TRONK

Desarrollo de sitio web de un entrenador personal que ofrece sus planes de entrenamiento y nutrición. Contiene 5 páginas.

Página "Planes" donde se pueden visualizar los planes ofrecidos y mediante un botón realizar la consulta a través de WhatsApp. Desde esta página también se puede acceder a una sección de preguntas frecuentes

Página "Contacto" tiene un formulario de contacto funcional conectado con Netifly para recibir consultas.

Tiene buena navegación y es responsive para todos los dispositivos, celular, tablet y PC (horizontal y vertical).

Tiene un buen % de SEO y accesibilidad.

Tecnologías utilizadas: HTML5, CSS3, Bootstrap.

MUY VOS SKIN

Desarrollo de sitio web para una dermocosmiatra y maquilladora que ofrece sus servicios y tratamientos. Contiene 3 páginas.

Página "Packs" donde el usuario puede elegir su tipo de piel para poder visualizar y comprar un pack recomendado.

Página "Tratamientos" donde puede visualizar los tratamientos corporales, faciales y makeup por separado y también realizar la compra de cualquiera de ellos.

Página "Contacto" validado con Javascript.

El "Carrito de compras" es funcional, al finalizar compra redirecciona a Mercadopago para simular la compra mediante link de pago

Tecnologías utilizadas: HTML5, CSS3, JavaScript y JQuery.

BAKERY

El proyecto simula un e-commerce de una tienda de pastelería. El sitio se compone por una página “Home” de presentación, una página "Nuestros Productos" con el catálogo completo y páginas según la categoría de productos.

Posee una página "Contacto" con un formulario funcional para enviar consultas, validado con Formik y Yup.

Página "Registro - Inicio de sesión" para que el usuario pueda Registrarse o Iniciar Sesión desde una cuenta ya creada o una cuenta de Google.

El proyecto utiliza Firebase "Firebase-Firestore" como base de datos para cargar todos los productos y control de stock. Y "Firebase-Authentication" para el registro e inicio de sesión.

Tecnologías utilizadas: React JS, JavaScript, HTML5, CSS3.
